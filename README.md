# Electrical Circuit Solver
Um solucionador de circuitos elétricos desenvolvido em Python com foco em fundamentos de Engenharia Elétrica.

## Sobre o Projeto
Este projeto foi criado com o objetivo de:
- Praticar lógica de programação
- Aplicar conceitos de Engenharia Elétrica
- Treinar organização de código
- Utilizar controle de versão com Git e GitHub
O programa funciona via terminal e permite realizar cálculos importantes em circuitos elétricos.

## Funcionalidades

### Lei de Ohm
- Cálculo de Tensão (V)
- Cálculo de Corrente (I)
- Cálculo de Resistência (R)

### Resistores
- Associação em Série
- Associação em Paralelo

### Potência Elétrica
- Cálculo usando V e I
- Cálculo usando R e I
- Cálculo usando V e R

## Como Executar
1. Certifique-se de ter o Python instalado (versão 3.10+ recomendada).
2. Clone o repositório: https://github.com/wesleyc12/electrical-circuit-solver.git
3. Acesse a pasta do projeto.
4. Execute o arquivo principal: python solver_main.py
